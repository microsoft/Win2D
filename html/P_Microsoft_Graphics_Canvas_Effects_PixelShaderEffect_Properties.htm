PixelShaderEffectProperties Property
[Experimental] Collection of properties configures the constant buffer that will be passed to the custom pixel shader.

Remarks

This API is marked as [Experimental], meaning it may be subject to change in future Win2D releases.

Properties are used to configure the constant buffer which is provided as an input when drawing with a custom shader. The keys and types of this collection are determined by the selected shader. There will be one entry for each variable defined by the shader.

Supported property types are Single, Int32, Boolean, Vector2, Vector3, Vector4, Matrix3x2, Matrix4x4, plus arrays of any of these types.

If you are building for Windows or Windows Phone 8.1, you must specify vector and matrix property values using the types from Microsoft.Graphics.Canvas.Numerics rather than System.Numerics. This is not an issue with UWP, where the System.Numerics types are automatically projected into WinRT.
