DistantDiffuseEffect Properties

The DistantDiffuseEffect type exposes the following members.

Properties
  NameDescription
Azimuth
Direction of the light source, measured in radians in the XY plane, counter-clockwise relative to the X axis. Default value 0, range 0 to 2*pi.
BufferPrecision
Specifies what precision to use for intermediate buffers when drawing this effect.
CacheOutput
Enables caching the output from drawing this effect.
DiffuseAmount
Intensity of the diffuse light. Default value 1.
Elevation
Elevation of the light source, measured in radians in the YZ plane, counter-clockwise relative to the Y axis. Default value 0, range 0 to 2*pi.
HeightMapInterpolationMode
Interpolation mode used to filter the height map. Default value Linear.
HeightMapKernelSize
Size of the Sobel filter kernel used to convert the input heightmap to surface normal vectors. Default value (1,1).
HeightMapScale
Height map scale factor in the Z direction. Default value 1.
LightColor
Color of the incoming light. Default value white.
LightColorHdr
[Experimental] High-dynamic-range color of the incoming light. Default value white.
Name
Attaches a user-defined name string to the effect.
Source
Gets or sets the input source for DistantDiffuse effect.
