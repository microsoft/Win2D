public byte[] GetPixelBytes()

Return Value

Type: Byte
Remarks
Works on bitmaps of any format. The size of the array is (Width of bitmap, in pixels) X (Height of bitmap, in pixels) X (bytes per pixel). The number of bytes per pixel is determined from CanvasBitmap's pixel format. For a CanvasBitmap with the default format of DXGI_FORMAT_B8G8R8A8_UNORM, for example, this is 4.
