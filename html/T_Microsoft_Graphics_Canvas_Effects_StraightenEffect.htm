Remarks

This API is available on Windows 10 (Universal Windows Platform) only. It is not supported on Windows 8.1 or Phone 8.1.

Supported by Win2D but not Windows.UI.Composition.

This effect is useful for correcting photos that were taken at an angle so their horizon is not quite horizontal.

This Windows Runtime type corresponds to the D2D Straighten effect.
