Remarks
Examples

Suppose, we have a cat image loaded as CanvasBitmap,

Copy
GaussianBlurEffect blurEffect = new GaussianBlurEffect()
{
    Source = cat,
    BlurAmount = 2.0f
};

Use the Draw event to display it:

Copy
void myWidget_Draw(CanvasControl sender, CanvasDrawEventArgs args)
{
    args.DrawingSession.Clear(Colors.CornflowerBlue);
    args.DrawingSession.DrawImage(blurEffect, new Point(42, 42));
}

This Windows Runtime type corresponds to the D2D Gaussian blur effect.
