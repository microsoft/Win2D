Remarks

This API is available on Windows 10 (Universal Windows Platform) only. It is not supported on Windows 8.1 or Phone 8.1.

Uses default flattening tolerance and identity transform on the input strokes.

Geometry created from ink typically contains many overlapping stroke segments and circles. After using CreateInk, it may be desirable to clean up these self-intersections by calling Outline Overload.

Note that CanvasGeometry does not support pencil ink strokes (InkDrawingAttributesKind.Pencil). Passing pencil strokes will produce an empty geometry.
