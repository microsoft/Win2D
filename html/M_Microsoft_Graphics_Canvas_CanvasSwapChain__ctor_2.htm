Remarks
The swap chain's pixel format will default to DirectXPixelFormat::B8G8R8A8UIntNormalized, its buffer count will be 2, and its alpha mode will be CanvasAlphaMode::Premultiplied. Size is in device independent pixels (dips), and DPI is taken from the specified resource creator interface.
