CanvasFontFace Properties

The CanvasFontFace type exposes the following members.

Properties
  NameDescription
Ascent
[Experimental] Gets the ascent value of the font face. Ascent is the distance from the top of font character alignment box to the Latin-script baseline.
CapHeight
[Experimental] Gts the cap height value of the font face. Cap height is the distance from the Latin-script baseline to the top of a typical Latin-script capital. Capital "H" is often used as a reference character for the purpose of calculating the cap height value.
CaretOffset
[Experimental] Gets the horizontal offset of the caret along the baseline for good appearance. An offset of 0 is returned for perfectly upright fonts (non-italic).
CaretSlopeRise
[Experimental] Gets the vertical rise of the caret. CaretSlopeRise / CaretSlopeRun yields the caret angle. A rise value of 1 is returned for perfectly upright fonts (non-italic).
CaretSlopeRun
[Experimental] Gets the horizontal run of the caret. CaretSlopeRise / CaretSlopeRun yields the caret angle. A run value of 0 for perfectly upright fonts (non-italic).
Descent
[Experimental] Gets the descent value of the font face in font design units. Descent is the distance from the bottom of font character alignment box to the Latin-script baseline.
FaceNames
[Experimental] Gets a map of localized strings that contain the face names for the font (for example, Regular or Bold).
FamilyNames
[Experimental] FamilyNames
FileFormatType
[Experimental] Gets the file format type of a font face.
GlyphBox
[Experimental] Gets the accumulated draw bounds of all glyphs in the font.
GlyphCount
[Experimental] Gets the number of glyphs in the font.
HasTypographicMetrics
[Experimental] Gets whether the ascent, descent, and lineGap are based on newer 'typographic' values in the font, rather than legacy values.
HasVerticalGlyphVariants
[Experimental] Gets whether the font has any vertical glyph variants.
IsMonospaced
[Experimental] Gets whether the font is monospaced; that is, all glyphs' layout boxes are equally sized and spaced apart.
IsSymbolFont
[Experimental] Gets whether the font is a symbol font.
LineGap
[Experimental] Gets the line gap for the font. This is the recommended additional white space to add between lines.
LowercaseLetterHeight
[Experimental] Gets the lowercase letter height of the font face. This is the distance from the Latin-script baseline to the top of lowercase letter "x", or a similar lowercase character.
Simulations
[Experimental] Gets the algorithmic style simulation flags of a font face.
Stretch
[Experimental] Gets the the degree to which the font is stretched, compared to the normal aspect ratio of that font.
StrikethroughPosition
[Experimental] Gets the strikethrough position, which is relative to the Latin-script baseline.
StrikethroughThickness
[Experimental] Gets the thickness of the strikethrough for this font.
Style
[Experimental] Gets whether the typeface is oblique, or italic, or the default.
SubscriptPosition
[Experimental] Gets the subscript position, which is relative to the baseline origin.
SubscriptSize
[Experimental] Gets the size of a subscript.
SuperscriptPosition
[Experimental] Gets the superscript position, which is relative to the baseline origin.
SuperscriptSize
[Experimental] Gets the size of a superscript.
UnderlinePosition
[Experimental] Gets the underline position, relative to the Latin-script baseline.
UnderlineThickness
[Experimental] Gets the thickness of underlines.
UnicodeRanges
[Experimental] Gets a list of character ranges supported by a font.
Weight
[Experimental] Gets the weight of this font.
