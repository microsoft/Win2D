Remarks

This API is only available when running on the Windows 10 November Update (build 10586 or greater).

The value of the tint parameter is a Vector4 that specifies the red, green, blue and alpha amounts to scale the bitmap's color by. The default value is Vector4.One. A value of Vector4(2, 1, 1, 1) would tint the sprite red by doubling the red amount of each source color. A value of Vector4(1, 1, 1, 0.5f) would draw the sprite at half opacity. The tint parameter is specified in non-premultiplied format.
